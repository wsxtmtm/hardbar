Подключить Spotify
Виджет «Сейчас играет» будет показывать реальный трек с твоего аккаунта — название, артист и обложка альбома в реальном времени.
Работает везде — не нужен HTTPS и сервер
1
Открой Spotify Console
Перейди по ссылке — это официальный инструмент разработчика Spotify для тестирования API:
Открыть Spotify Console
2
Получи токен
На странице Console нажми кнопку «Get token». Отметь галочки:
user-read-currently-playing и user-read-playback-state.
Нажми «Request token» → войди в свой Spotify аккаунт.
Токен действует 1 час. Когда истечёт — повтори процедуру.
3
Вставь токен сюда
Скопируй длинную строку из поля «OAuth Token» и вставь ниже.
Токен слишком короткий — убедись, что скопировал полностью
Требует HTTPS — сначала разверни сайт на Netlify или другом хостинге
1
Разверни сайт с HTTPS
Перетащи папку с сайтом на app.netlify.com/drop — получишь бесплатный HTTPS-адрес за 30 секунд.
2
Создай Spotify App
На developer.spotify.com/dashboard создай приложение, в Edit settings → Redirect URIs добавь этот адрес:
Открой эту страницу через HTTPS чтобы увидеть URI
3
Введи Client ID
Client ID находится на главной странице твоего приложения в Spotify Dashboard.
Введи корректный Client ID
Подключаем аккаунт...
Spotify подключён
